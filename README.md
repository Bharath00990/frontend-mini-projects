# Basic Front-End Projects

## Overview

This repository contains simple front-end projects developed using HTML, CSS, and JavaScript. These projects demonstrate API integration, responsive UI design, and DOM manipulation.

## Projects

### 🌤️ Weather App

A weather application that fetches real-time weather information using WeatherAPI.

#### Features
- Search weather by city name
- Displays:
  - Temperature
  - Weather condition
  - Humidity
  - Wind speed
- Responsive and modern interface

#### Technologies Used
- HTML
- CSS
- JavaScript
- WeatherAPI

---

### 🎬 Movie Search App

A movie search application that retrieves movie details from OMDb API.

#### Features
- Search movies by title
- Displays:
  - Movie poster
  - Title
  - Release year
- Direct IMDb link for each movie
- Responsive card layout

#### Technologies Used
- HTML
- CSS
- JavaScript
- OMDb API
